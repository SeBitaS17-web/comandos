git init:Este comando inicializa un nuevo repositorio Git en el directorio actual. Crea una carpeta oculta llamada .git que contiene todos los archivos necesarios para el seguimiento del proyecto.

git add:Agrega cambios en los archivos al área de preparación (staging area). Esto significa que Git empieza a rastrear esos cambios para el próximo commit.
